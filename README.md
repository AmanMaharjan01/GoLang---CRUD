# CRUD-GO

This project is a simple CRUD application with 3-tier architecture using GoLang and PostgreSQL as DB.

## Prerequisites

- Go
- Postgres

## Run Application

Create .env and set `PORT=8000` and `POSTGRES_URL={url}` and run

```bash
go run main.go
```
